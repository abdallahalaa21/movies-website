## How to run the code

## 1-clone the repo
`git clone git@github.com:abdallahalaa21/movies-website.git`

## 2-Install the packages
you need to install nodeJs first then use npm 
<br />
or install yarn after that and use it
<br />
`yarn` or `npm install`

## 3-Run the code
`yarn start` or `npm run start`
